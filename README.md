# 🎬 FilmDex

A sleek movie discovery app powered by the [TMDB API](https://www.themoviedb.org/), built with React + Vite.

![FilmDex Preview](https://image.tmdb.org/t/p/original/placeholder.jpg)

## Features

- 🔍 **Live search** — debounced search across the entire TMDB catalogue
- 🎬 **Categories** — Popular, Top Rated, Now Playing, Upcoming
- 🦸 **Hero banner** — cinematic spotlight on the top movie
- ♥ **Save movies** — favourite any movie and view your saved list
- 📄 **Pagination** — browse up to 20 pages of results
- 📱 **Fully responsive** — 2–6 column grid across all screen sizes
- ✨ **Glassmorphism UI** — dark purple aesthetic with glass cards

## Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [TMDB API](https://developer.themoviedb.org/docs)

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/DakshSawant/FilmDex.git
cd FilmDex
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

### 4. Build for production

```bash
npm run build
```

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push to GitHub
2. Import repo on [vercel.com](https://vercel.com)
3. Framework preset: **Vite**
4. Deploy — done ✅
